# Booking System

예약 관리 시스템

## 개발 환경 설정

```bash
git clone https://github.com/YonghaSong/booking-system.git
cd booking-system
npm install
```

## 실행

```bash
node src/index.js
```

## 프로젝트 구조

```
booking-system/
├── src/           # 소스 코드
├── tests/         # 테스트 파일
├── docs/          # 문서
└── README.md
```
